# Contributing to Decktype

Thanks for contributing! Decktype is a playful, experimental typing project, so bold ideas are welcome.

## Quick start

```bash
bun install
cd apps/frontend && bun dev
cd apps/backend && bun dev
```

## Guidelines

- Keep changes focused and small when possible.
- Prefer clear, readable code over cleverness.
- If you change behavior or add a feature, update docs accordingly.

## Submitting a PR

1. Fork the repo and create a feature branch.
2. Make your changes.
3. Open a PR describing **what** you changed and **why**.

## Code of conduct

Be respectful, assume positive intent, and keep feedback constructive.
